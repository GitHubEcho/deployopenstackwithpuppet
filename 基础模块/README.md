
# 基础模块

何为基础模块？基础模块对应的是Openstack中所使用到的基础服务，例如数据库，消息队列，缓存，Web服务器等等，或者是操作系统的配置模块，例如防火墙，网卡等。

本书的主旨是讲解Openstack的自动化部署，因此在这些模块中，我们只会去说明Openstack使用到的class或者define。

* puppet-apache
* puppet-corosync  
* puppet-firewall 
* puppet-memcached 
* puppet-mongodb 
* puppet-mysql 
* puppet-rabbitmq 
* puppet-rsync
* puppet-sysctl
* puppet-vswitch
* puppet-xinetd
